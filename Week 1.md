
# MATLAB 첫주차

```
… → 다음 엔터에서도 입력가능.

; → 입력해도 출력물을 안보여줌

clear all → Workspace 없애줌 (다 날라감)

% → 주석 같은거 옆에 설명

clc -> command window 없애줌 

command History -> 눌러서 예전거 불러올 수 있음

\ -> 사칙연산 다 똑같은데 왼쪽 나눗셈이 존재 (ex 3\5 = 5/3)

사칙연산 순서 ()(Parenthesis) -> ^(Exponentiation) -> *(Multiplication) / -> +(addition) -(subtraction) -> 왼쪽부터

sqrt(x) = squared root x
nthroot(x,n) -> n root x = ^(1/n) ex) nthroot(0.18,3) = 0.18^(1/3)
exp(x) = e^x
abs(x) = absolute value
log(x) = ln x
log10(x) 
factorial(x) = x!

Trigonometric Functions

sin(x), sind(x)-> degree asin(x), asind(x)-> degree
cos(x), cosd(x)-> degree acos(x), acosd(x)-> degree
tan(x), tand(x)-> degree atan(x), atand(x)-> degree
cot(x), cosd(x)-> degree acot(x), acosd(x)-> degree

Rounding Functions

round(x) -> 반올림
fix(x) -> 소수점 잘라버림
ceil(x) -> 가장 가까운 작은 정수로 (올림) ->
floor(x) -> 가장 가까운 큰 함수로 (버림) -> 두개 헷갈리니까 찾아보기
rem(x,y) -> x를 y로 나누었을때 나머지
sign(x) -> x>0 이면 1 <0이면 -1 0이면 0 반환

Rule for Variable name

Complex number
3+4i
```
